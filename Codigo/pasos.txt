Crear entorno virtual:
    -> python -m venv myenv

Activar entorno virtual:
    -> myenv\Scripts\activate

Instalar:
    -> pip install django==5.0
    -> pip install djangorestframework

Crear proyecto django y moverse a la carpeta del proyecto:
    -> django-admin startproject myproject
    -> cd myproject
    -> pip install mysqlclient

Crear una BD, ir a myproject/settings.py y configurar el apartado "DATABASES"
    DATABASES = {
        'default': {
            'ENGINE': 'django.db.backends.mysql',
            'NAME': 'mydatabase', #Nombre de la base de datos creada
            'USER': 'root',       # Usuario por defecto en XAMPP
            'PASSWORD': '',       # Contraseña vacía por defecto
            'HOST': 'localhost',  # O '127.0.0.1'
            'PORT': '3306',
        }
    }

Crear las aplicaciones:
    -> python manage.py startapp nombreApp

Registrar las apps myproject/settings.py en el apartado "INSTALLED_APPS"
    INSTALLED_APPS = [
        ...
        'rest_framework',
        'nombreApp',
    ]

Crear los Models de cada app creada

Realizar las migraciones:
    -> python manage.py makemigrations
    -> python manage.py migrate

Crear los serializers (se debe crear el archivo serializers.py en cada app):
    -> Sino detecta el rest_framework seguir estos pasos:
        Abrir la paleta de comandos (Ctrl+Shift+P)
        Buscar "Python: Select Interpreter"
        Seleccionar el intérprete que está en la carpeta venv/ o donde este el espacio de trabajo

Crear las views para cada app en views.py

Crear un superusuario:
    -> python manage.py createsuperuser

Levantar el servidor
    -> python manage.py runserver
    -> Si todo funciona se verá esto: Starting development server at http://127.0.0.1:8000/
    -> Ir a http://127.0.0.1:8000/admin e ingresar con los datos del superusuario

Hacer endpoints básicos con POSTMAN
    -> http://127.0.0.1:8000/api/pacientes/
    -> http://127.0.0.1:8000/api/medicos/


Comandos importantes:
    Para borrar todas las migraciones:
        -> find . -path "*/migrations/*.py" -not -name "__init__.py" -delete
        -> find . -path "*/migrations/*.pyc"  -delete
